# Titanic Survival Prediction Portfolio

이 프로젝트는 타이타닉 탑승객 데이터를 활용하여
생존 여부를 예측하는 머신러닝 모델을 개발한 포트폴리오입니다.

## 프로젝트 개요
- 목표: 탑승객의 생존 여부 예측
- 데이터: Kaggle Titanic Dataset
- 결과: RandomForest 모델 기준 약 83% 정확도 달성

## 사용 기술
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## 주요 작업 내용
- 데이터 정제 및 결측치 처리
- EDA 시각화 분석
- 파생 변수 생성
- 모델 학습 및 평가

## 결과물
- 분석 코드 (Notebook)
- 시각화 이미지
- 모델 결과 요약 자료
